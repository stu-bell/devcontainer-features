# Dev Container Features

This repo is a collection of dev container features. Follow the links below for documentation on each feature:

- [Cursor CLI](./src/cursorcli) - AI code assistant [cursor.com/cli](https://cursor.com/cli)
- [Gemini CLI](./src/geminicli) - AI code assistant [geminicli.com](https://geminicli.com)
- [Neovim](./src/neovim) - Neovim editor, with config via Git [neovim.io](https://neovim.io)
- [Node.js](./src/node) - Node.js - this is a dependency feature only
